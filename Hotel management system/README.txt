# Hotel Management System - Python GUI

# Created By Mario Agbanobi

This project is a simple **Hotel Management System** created using **Python** and **Tkinter** for the GUI. It allows users to check-in and check-out guests, display room prices, and manage food services during a guest's stay.

## Features

- **Room Management**: Displays the status of rooms (Available/Occupied).
- **Check-in Process**: 
  - Enter room number and guest name.
  - Displays room price and provides an option to include food service.
  - Calculates the total bill based on room price and food service.
- **Check-out Process**: 
  - Allows guests to check-out and updates the room status to "Available".
  - Displays the bill at check-out.
- **Food Service Option**: Option to add food service for an additional fee of `$30`.
- **Dynamic UI**: The interface dynamically updates room availability and guest information.

## Requirements

To run the project, you'll need:

- Python 3.x
- Tkinter (Usually pre-installed with Python)
